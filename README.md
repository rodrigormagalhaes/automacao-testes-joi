### Automação de testes de contrato de API com Joi

- **Pré requisito**
  > [Node.js](https://github.com/nodesource/distributions/blob/master/README.md#debinstall) 

- **Instalar**:
```
npm install
```

- **Executar testes**:
```
npm test
npm run test1
```

- **Relatório html**: 
  > ./mochawesome-report/mochawesome.html
